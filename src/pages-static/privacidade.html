Política de Privacidade - AssistJur.IA

Última atualização: 18 de setembro de 2025

1. Coleta de Dados

Coletamos apenas os dados necessários para fornecer nossos serviços de análise jurídica.

2. Uso dos Dados

Os dados são utilizados exclusivamente para análises processuais e melhorias da plataforma.

3. Compartilhamento de Dados

Não compartilhamos dados pessoais com terceiros, exceto quando exigido por lei.

4. Segurança

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados.

5. Seus Direitos

Você tem direito de acesso, correção, exclusão e portabilidade dos seus dados pessoais.

6. Contato

Para questões sobre privacidade, entre em contato através do site oficial.
